## Running the app

```bash
# development access at localhost:3000
$ npm run start:dev

# build and run production mode access at localhost:3000
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Lint
```bash
# run prettier and linter
$ npm run lint
```
