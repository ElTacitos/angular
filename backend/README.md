## Running the app

```bash
# access at localhost:3000
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# endpoints tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Lint
```bash
# run prettier and linter
$ npm run lint
```
